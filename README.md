# شركة الحكمة للتجهيزات الطبية (Al Hikma Medical)

موقع إلكتروني احترافي مخصص لشركة الحكمة للتجهيزات الطبية في سوريا، تم تطويره باستخدام React و Vite و Tailwind CSS.

## كيفية الرفع على Vercel

لرفع هذا الموقع على Vercel، اتبع الخطوات التالية:

### 1. إعداد المستودع (Repository)
- قم برفع ملفات المشروع إلى GitHub.
- تأكد من تضمين ملف `vercel.json` الموجود في المجلد الرئيسي.

### 2. الربط مع Vercel
- ادخل إلى [Vercel Dashboard](https://vercel.com/dashboard).
- اضغط على **"Add New"** ثم **"Project"**.
- اختر المستودع الخاص بك.
- في إعدادات المشروع (Project Settings):
  - **Framework Preset**: اختر **Vite**.
  - **Build Command**: اتركه افتراضياً (غالباً `npm run build`).
  - **Output Directory**: تأكد من ضبطه على `dist`.
- اضغط على **"Deploy"**.

### 3. إعداد الدومين (Domain)
- بعد اكتمال عملية الـ Deploy، سيظهر لك رابط مؤقت (مثل `alhikma.vercel.app`).
- للربط مع `alhikma.com`:
  - اذهب إلى إعدادات المشروع في Vercel.
  - انتقل إلى تبويب **"Domains"**.
  - أدخل اسم النطاق الخاص بك `alhikma.com`.
  - اتبع التعليمات لتحديث سجلات الـ DNS في لوحة تحكم شركة تسجيل الدومين (مثل GoDaddy أو Namecheap).
